# Basic C++ Programs

Welcome! This repository is a growing collection of simple, beginner-friendly C++ programs. 
Each program here is designed to help new programmers understand fundamental C++ concepts through straightforward code examples. 
If you are just starting with C++, you'll find these examples helpful for learning and practice.
As I continue learning, I’ll add more programs covering different topics.

## Contents

- `add-two-numbers.cpp` — Adds two numbers and prints the result.
- `positive-negative numbers checker.cpp` _ check whether a number is positive,negative or zero.
 `2player.cpp`_one player enters a number and the other guesses it.
 `simple-5operation-calculator.cpp` _takes input from user and performs a specific operation
More programs will be added soon!

---

**How to run:**
1. Download or clone the repo.
2. Compile with a C++ compiler, for example:
    ```dev-cpp or vs-code
    g++ add-two-numbers.cpp -o add-two-numbers
    ./add-two-numbers
    ```
